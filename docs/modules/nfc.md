# NFC Module

## Purpose

The NFC module supports reading and writing NFC tags for field workflows.

## Planned functions

- read text from NFC tag
- write text to NFC tag
- read JSON context from NFC tag
- write ODK launch data to NFC tag
- return tag payload to ODK

## Categories

- Tags
- Forms
- Utilities

## Return modes

- Fields
- JSON

## Example use cases

- participant card
- household tag
- sample container tag
- location marker
- device pairing
