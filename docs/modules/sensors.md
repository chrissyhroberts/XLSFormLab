# Sensor Modules

## Purpose

Sensor modules expose Android device sensors and external sensor workflows to ODK.

## Planned modules

- GPS point
- compass bearing
- accelerometer
- light meter
- sound meter
- speed measurement
- device orientation
- BLE device proximity

## Return modes

Sensor modules should support JSON for rich measurements and field returns for simple values.

## Considerations

- permissions
- calibration
- hardware availability
- confidence and accuracy metadata
- repeated measurements
