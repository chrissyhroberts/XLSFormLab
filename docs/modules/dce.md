# DCE Module

## Purpose

The DCE module supports discrete choice experiment tasks inside ODK workflows.

## Planned task types

- binary choice
- multinomial choice
- best-worst choice
- ranking
- adaptive choice

## Return values

- selected alternative
- task id
- attribute set
- response time
- optional confidence rating

## Configuration

DCE tasks may require JSON configuration because they can be too complex for simple field-level intent extras.
