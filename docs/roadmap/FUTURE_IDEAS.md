# Future Ideas

## Capability explorer

A self-documenting catalogue of installed modules with example intents, schemas and return values.

## Form designer

A visual tool that lets users configure a capability and copy an XLSForm snippet.

## Profiles

Named JSON bundles of settings.

Examples:

- VAS pain scale
- WHO disability scale
- contact tracing household roster
- sample chain-of-custody

## Plugin ecosystem

Future versions may support externally distributed capabilities.

## Beyond ODK

The Capability Engine should remain useful for other data collection platforms that can invoke Android intents or structured APIs.
